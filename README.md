# Oikos Energy - Website Oficial

Website profesional futurist pentru Oikos Energy România (oikosenergy.ro)

## Tehnologii

- Next.js 15.2 (Pages Router)
- React 18.3
- TypeScript
- Tailwind CSS v3
- Supabase (Backend & Autentificare)
- shadcn/ui Components

## Deployment

Proiectul este configurat pentru deployment pe Vercel cu următoarele variabile de mediu:
- NEXT_PUBLIC_SUPABASE_URL
- NEXT_PUBLIC_SUPABASE_ANON_KEY
- SUPABASE_SERVICE_ROLE_KEY

## Dezvoltare Locală

```bash
npm install
npm run dev
```

Website-ul va fi disponibil la http://localhost:3000
